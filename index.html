Golden Sun Engine - HTML5 (MIT License)
Under development.

Use the arrows to move.
Hold shift to dash.
1,2 and 3 for zoom.
Double click on screen to fullscreen.
D for debug colision. F for fps. G for grid. K for keys debug. S for battle stats. L for sliders.
Use the enter key to talk to npc.
Use the spacebar to open the menu. Use ESC and ENTER to navigate.
Go to east for pillar test. Go west for Madra town.
Use Q to cast Move. Use W to cast Frost. Use E to cast Growth.
It can take a while to load. I'm not compressing the source for now.
Prefer to use Chrome instead of Firefox. Chrome is running better.

See the source code on github.
